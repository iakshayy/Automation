# Project Overview

- Manual Testing
- Test Case Design
- Bug Reporting
- Selenium Automation Testing

Application Under Test:
https://sauce-demo.myshopify.com/

# Automated Test Scenarios

The following scenarios are automated using Selenium with Java:

## 1. Successful Login
- User logs into the application using valid credentials.

## 2. Add Product to Cart
- Logged-in user adds product to cart successfully.

## 3. Successful Checkout
- Logged-in user proceeds to checkout successfully.
